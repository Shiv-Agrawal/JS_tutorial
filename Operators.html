Operators means the symbols that are used to do math. We can use them to add, subtract, multiply, divide, etc. variables. We use them mostly in things where we need to set a limit or do some math to the data fetched or we have written.

Operators Description
+ Used in addition
- Used in subtraction
* Used to multiply
/ Used to divide
** Used in exponentiation
% Used to find modulus (division remainder)
++ Used to increment
-- Used to decrement
